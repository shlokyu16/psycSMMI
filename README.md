## Overview

This project investigates the measurable relationship between social media usage patterns and mental health risk indicators — specifically ADHD-related attentional difficulties, anxiety, self-esteem disruption, and depressive symptoms. A custom-weighted scoring pipeline transforms Likert-scale survey responses into composite domain scores, which are then used to train and evaluate a suite of machine learning classifiers.

The primary output is a deployed Streamlit application that allows users to complete a 13-question survey and receive a real-time binary mental health risk prediction from a trained MLP Neural Network (~98.9% test accuracy).

---

## Research Objectives

1. Quantify correlations between social media behavior and four mental health domains: ADHD, Anxiety, Self-Esteem, Depression
2. Design and justify a custom non-linear scoring system that captures sub-clinical signal from Likert-scale responses
3. Comparatively evaluate 8+ machine learning models under identical preprocessing and train/test conditions
4. Deploy the best-performing model in an accessible, interactive web interface

## Methodology

### Dataset

- **Source:** [Social Media and Mental Health — Kaggle](https://www.kaggle.com/datasets/souvikahmed071/social-media-and-mental-health)
- **Size:** 481 valid responses (3 removed: non-binary gender entries with insufficient representation)
- **Features:** 20 survey questions covering demographics, social media usage frequency, and 12 Likert-scale mental health indicators
- **Scope:** Primarily university-aged respondents (18–24); demographics include sex, relationship status, and occupation

### Custom Weighted Grading System

A core methodological contribution of this work is the non-linear remapping of standard 1–5 Likert responses. Rather than using raw ordinal values, each response is transformed as follows:

**Standard questions (ADHD Q1–Q4, Anxiety Q1–Q2, Self Esteem Q1, Q3, Depression Q1–Q3):**

| Likert Response | Meaning            | Weighted Score |
|----------------:|--------------------|:--------------:|
| 1               | Very Positive      | -1             |
| 2               | Slightly Positive  | 0              |
| 3               | Neutral            | 0              |
| 4               | Slightly Negative  | 3              |
| 5               | Very Negative      | 5              |

**Self Esteem Q2** ("How do you feel about comparisons to others?") uses an **inverted scale** in the original survey (1 = Very Negative, 5 = Very Positive). To ensure higher scores consistently denote greater risk across all questions:

| Likert Response | Meaning            | Weighted Score |
|----------------:|--------------------|:--------------:|
| 1               | Very Negative      | 5              |
| 2               | Slightly Negative  | 3              |
| 3               | Neutral            | 0              |
| 4               | Slightly Positive  | -2             |
| 5               | Very Positive      | -4             |

**Rationale:** Neutral-to-positive responses are zeroed out to reduce noise from non-committed answers. Moderately negative responses (Likert 4) receive a disproportionate weight of 3, reflecting the clinical observation that consistent mild-negative symptom endorsement is more diagnostically meaningful than isolated extreme responses.

### Domain Score Aggregation

Individual question scores are summed into four composite domain scores:

- **ADHD Score** = ADHD Q1 + Q2 + Q3 + Q4 (max: 20, min: −4)
- **Anxiety Score** = Anxiety Q1 + Q2 (max: 10, min: −2)
- **Self Esteem Score** = SE Q1 + Q2 + Q3 (max: 15, min: −6)
- **Depression Score** = Depression Q1 + Q2 + Q3 (max: 15, min: −3)
- **Total Score** = Sum of all four domain scores

### Binary Outcome Variable & Threshold Selection

The binary outcome (0 = No Risk, 1 = At Risk) is derived from Total Score using a threshold of **22.5**. Selection rationale:

1. **Data-driven initial estimate:** KMeans (k=2) clustering on Total Score, followed by Youden's J statistic on the resulting ROC curve, yields an optimal cutoff of ~18.0. At this threshold, ~44% of respondents are classified as at-risk.
2. **Epidemiological calibration:** WHO (2022) prevalence estimates for combined anxiety and depression in young adults are approximately 30–37%. The cutoff of 22.5 produces a ~37% at-risk rate, aligning the outcome distribution with the benchmark.

### Feature Selection

Pearson correlation analysis confirms that demographic features (Age, Sex, Relationship Status, Occupation) have near-zero correlation with Outcome (|r| < 0.08), indicating that social media's effect on mental health risk is largely demographic-invariant in this dataset. These features are excluded from model training.

**Final model input features:** Time Spent (encoded 0–5), ADHD Score, Anxiety Score, Self Esteem Score, Depression Score

### Model Comparison

Nine models were evaluated under identical conditions (stratified 80/20 split, `random_state=16`):

| Model               | Hold-out Accuracy | CV Accuracy (10-fold) | Notes                         |
|---------------------|:-----------------:|:---------------------:|-------------------------------|
| Logistic Regression | ~100.0%           | ~98% ± 2%             | Likely linear separability artefact |
| **MLP Neural Network** | **98.9%**      | **~97% ± 2%**         | **Selected for deployment**   |
| SVC                 | ~97.9%            | ~97% ± 2%             | Valid alternative              |
| FNN (Keras)         | ~97.9%            | —                     | Heavier dependency stack       |
| SGD                 | ~96.8%            | varies                | Less stable across folds       |
| KNN                 | ~96.8%            | varies                | No probability output          |
| Gaussian NB         | ~95.8%            | varies                | Useful baseline                |
| Random Forest       | ~94.7%            | varies                |                                |
| Decision Tree       | ~88.4%            | varies                | Overfits individual splits     |

**Selected model:** `MLPClassifier(solver='lbfgs', alpha=1e-5, hidden_layer_sizes=(5120, 16))` — chosen over LR (artefact-prone on small datasets) and SVC (equivalent performance, but MLP serialises cleanly with joblib for lightweight Streamlit deployment without TensorFlow dependency).

## Results

The MLP Neural Network achieves:
- **Test Accuracy:** ~98.9%
- **ROC-AUC:** reported in `final.ipynb`
- **Full metrics:** Precision, Recall, F1, Confusion Matrix — see `final.ipynb` output

Key finding: Time Spent on social media shows a moderate positive correlation with the Outcome variable, confirming that increased social media consumption is associated with elevated mental health risk across all four measured domains.

## Limitations

- **Dataset size:** ~480 responses limits generalisability. A larger, more demographically diverse dataset would strengthen reliability.
- **Self-report bias:** Likert-scale surveys are susceptible to social desirability bias.
- **Binary classification:** Mental health risk is a spectrum; a binary label is a simplification. Future work could explore multi-class or regression approaches.
- **Cross-sectional design:** The dataset captures a single point in time; longitudinal data would enable causal inference.
- **Demographic scope:** The dataset skews toward university-aged respondents and may not generalise to older populations.
